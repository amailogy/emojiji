# emojiji - 絵文字×時事

日替わり時事風刺ゲーム。毎日出題される時事ワードに対して、絵文字だけで風刺アートを作り、Xに共有します。

## ディレクトリ構成

```
emojiji/
├── index.html        ← アプリ本体（HTML/CSS/JS全部入り）
├── topics.json       ← お題データベース
├── add_topic.sh      ← お題追加用シェルスクリプト
└── README.md
```

## デプロイ

静的ファイルのみで動作します。以下のいずれかにそのままデプロイ可能：

- **GitHub Pages**: リポジトリ設定 → Pages → Source を `main` ブランチに設定
- **Vercel**: `vercel` コマンドでデプロイ、またはGitHub連携
- **Netlify**: ディレクトリをドラッグ＆ドロップ、またはGit連携

## お題の追加方法

### 手動編集

`topics.json` を直接編集：

```json
{
  "2026-03-02": {
    "word": "AI失業",
    "desc": "人工知能の発展で消えゆく仕事。技術の進歩は恩恵か脅威か。",
    "hints": ["🤖","💼","😰","📉","🏭","👷","💻","🔧","❌","🧠","⚡","🚫"]
  }
}
```

### シェルスクリプト

```bash
./add_topic.sh 2026-04-01 "エイプリルフール" "嘘が許される日。" "🤥,😂,📱,🎭,❓,🤡,📰,💬,✨,😈,🔔,🎪"
```

引数: `<日付> <ワード> <説明> <カンマ区切り絵文字12個>`

### GitHub Actions 自動化

毎朝8時にニュースAPIからトピックを取得し、`add_topic.sh` で追加→自動コミットするワークフローを設定可能。

## アプリ仕様

- **1日1回制限**: 共有ボタン押下でlocalStorageに記録、同日再プレイ不可
- **Day番号**: 2026年3月2日を Day 1 として日付差分で計算
- **キャンバス操作**: タップ配置、ドラッグ移動、ピンチ拡大縮小、ゴミ箱削除
- **共有**: 5×5グリッドにスナップ、空セルは⬜で埋め、お題は非公開
- **iPhone最適化**: fixed配置、ズーム防止、safe-area対応
